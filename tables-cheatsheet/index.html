Tables

by oscarmires

Feb 12, 2021

The HTML ⟨table⟩ is an element that enables two-dimensional representation of tabular data in HTML pages.

Used tags

Tag Name Purpose Attributes
⟨table⟩...⟨/table⟩ Table Wraps the whole table
⟨tr⟩...⟨/tr⟩ Table row Wraps around a set of elements, defining them as belonging to the same row colspan, rowspan, headers
⟨th⟩...⟨/th⟩ Table header Defines a header for a column colspan, rowspan, scope
⟨td⟩...⟨/td⟩ Table data Marks the actual bits of data
⟨col⟩...⟨/col⟩ Column Represents nth column in ⟨colgroup⟩
⟨colgroup⟩...⟨/colgroup⟩ Column group Specifies styling information for columns

Styling

The following list presents some css properties that can enhance table styling:

border-collapse
Applied to: ⟨table⟩
When value is collapse, all cells in a table share borders.
When value is separate, cells don't share borders.

Sample code

    ⟨table⟩
        ⟨tr⟩
            ⟨th⟩Multiples of two⟨/th⟩
            ⟨th⟩Multiples of three⟨/th⟩
        ⟨/tr⟩
        ⟨tr⟩
            ⟨td⟩2⟨/td⟩
            ⟨td⟩3⟨/td⟩
        ⟨/tr⟩
        ⟨tr⟩
            ⟨td⟩4⟨/td⟩
            ⟨td⟩6⟨/td⟩
        ⟨/tr⟩
        ⟨tr⟩
            ⟨td⟩6⟨/td⟩
            ⟨td⟩9⟨/td⟩
        ⟨/tr⟩
    ⟨/table⟩

The previous code generates this table:

Multiples of two Multiples of three
2 3
4 6
6 9
